# ServiceStore-Web
Plataforma para modernização de assistência técnica mobile e venda de acessórios. Focada em otimizar o fluxo de atendimento, facilitar a localização física da loja via Google Maps e aumentar a autoridade digital através de estratégias de avaliação e fidelização de clientes. Solução prática para gestão de serviços e vitrine eletrônica.
